#  Web Protfolio Home Page

This website is hosted on an Apache server running on an AWS Linux EC2 instance. It features a static HTML and CSS page structure, with JavaScript used to dynamically update the inner HTML on click events.

##  Table of Contents

- [Features](#Features)
- [TechnologiesUsed](#TechnolgiesUsed)

# Features

1. **UI**
    - Seamlessly re-renders the static page using JavaScript on click events.
    - Utilizes innerHTML to dynamically change the content of the page.
    - Custom CSS for visual design and styling.

1. **Languages**
   - Javascript
   - HTML
   - CSS
3. **Tools**
   - Apache
   - AWS Linux EC2 Instance
   - Git
